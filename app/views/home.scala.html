Line Learning

This is a simple application to help you learn your lines.

To start, first create a Scene. Click Scene from the menu above and select the Add... button.

Once you have added a scene, you can add Cue Lines to it. Select the scene from the list using its name. Then select the Add... button. Enter a cue, then your first line following the cue, and press Save changes.

To test your lines, click the Run button either from the Cue lines page or from the Scenes page.

Top tip. To keep things simple, don't add punctuation!
